Choose Wallet

ConfluxPortal#

ConfluxPortal can be used for Trust Domains to access the Conflux Name Service in Conflux network.

MetaMask#

MetaMask can be used for decentralized applications in Ethereum, Binance Smart Chain and HecoChain.

If you want use MetaMask for Biance Smart Chain, you can follow MetaMask Guides - Binance Chain Docs to configure it for BSC network.

By use the MetaMask for HecoChain, you can follow HecoChain Docs to configure it for Huobi HecoChain network.

Binance Chain Extension Wallet#
