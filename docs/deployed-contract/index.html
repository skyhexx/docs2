Deployed Contract

Trust Domains deployed on the following blockchain networks.

Ethereum#

Mainnet#

Registry - 0xdef10006363548e9c181fad2976142b7947d9070

Resolver - 0xda5f50d64ca89936af472c421be7e645789c9916

Tether USD - 0xdac17f958d2ee523a2206206994597c13d831ec7

Tether USD is the USDT official USD based token deployed on Ethereum network by Tether Operations Limited. You can use the USDT to join the DeFi Name Service(.defi) auction service which deployed on Ethereum main network.

Kovan#

Registry - 0xdef10006363548e9c181fad2976142b7947d9070

Resolver - 0xda5f50d64ca89936af472c421be7e645789c9916

TrustUSDT - 0x5c66Afd369c4443e559D490cc2d9557377391B30

TrustUSDT is a test USDT for the TrustDomains. You can send any value of Kovan ETH to the token address of TrustUSDT, your will got 10000USDT automatically. The you can buy the test domains and test the auction service.

Conflux#

Mainnet#

Registry - cfx:acdzztjdv0vcx2z2znt296y6sd4hujswean18n0gxy

Resolver - cfx:acet0jf7t2ervz3xhtu1tzpe3md1xnumhjbud0t0yb

Conflux cUSDT - cfx:acf2rcsh8payyxpg6xj7b0ztswwh81ute60tsw35j7

Conflux cUSDT is a Conflux-peg USDT for the Tether USD. You can use the cUSDT to buy the Conflux Name Service(.cfx) domains and the join the auction service which deployed on Conflux main network.

Binance Smart Chain#

Mainnet#

Registry - 0x0000F22FfE0866FFB8834600DAd9259cF4956853

Resolver - 0xFA98282DD3511144758B9D550E31fb5a4428B5cE

BUSDT - 0x55d398326f99059ff775485246999027b3197955

BUSDT is Binance-Peg BUSD-T. You can use the BUSDT to buy the BSC Name Service(.bnb) domains which deployed on BSC main network and join the auction of the .bnb domains.

HecoChain#

Mainnet#

Registry - 0x00072D9DcEC3FCFb80a60286C23d1fd8E3a7B5F5

Resolver - 0x72B0569A2263f5851EA5c8f7E1EA5B3347b99B5C

HUSDT - 0xa71edc38d189767582c38a3145b5873052c3e47a

HUSD is Heco-Peg USDT Token. You can use the HUSDT to buy the Heco Name Service(.heco) domains which deployed on HecoChain main network and join the auction of the .heco domains.
