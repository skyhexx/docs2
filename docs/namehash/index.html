Namehash

Namehash is an recursive algorithm generate a unique ERC-721 token id for the human-readable names. The Trust Domains are an ERC721 compliant nonfungable tokens, you can transfer your domains by use the wallet which support NFTs. Also, all domains can transferred use the domain name with the extended ERC-721 token methods.

Algorithm#

Namehash is specified in EIP-137. A sample implementation in Python is provided below.

def namehash(name):
if name == '':
return '\0' * 32
else:
label, _, remainder = name.partition('.')
return sha3(namehash(remainder) + sha3(label))

Three level domains#

The blockchain domains designed to replace the cryptocurrency addresses with the human-readable names, and the length of the names is the shorter the better, so Trust Domains restrict the format only supported three levels.
